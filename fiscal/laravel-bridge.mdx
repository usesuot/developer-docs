## Instalação

```bash
composer require suot/fiscal-laravel
php artisan vendor:publish --tag=fiscal-config
```

Configure `FISCAL_ENVIRONMENT`, `FISCAL_CERT_PATH`, `FISCAL_CERT_PASSWORD`, `FISCAL_ISSUER_UF`. Produção exige `FISCAL_ALLOW_PRODUCTION=true`.

## Injetar `Cte` (padrão)

```php
use Suot\Fiscal\Cte\Application\Cte;
use Suot\Fiscal\Core\Exception\InfrastructureException;
use Suot\Fiscal\Core\Result\RetryMode;

final class EmitirCteAction
{
    public function __construct(
        private readonly Cte $cte,
    ) {}

    public function __invoke(): void
    {
        try {
            $result = $this->cte
                ->issue()
                ->forOperation('transport-order-123')
                ->inHomologation()
                // … cadeia tipada …
                ->handle();
        } catch (InfrastructureException $e) {
            $failure = $e->toFailure('issue');
            // Host: alertar; filar retry conforme $failure->retryPolicy (timeout ambíguo → QueryBeforeRetry)
            throw $e;
        }

        if ($result->isAuthorized()) {
            // Host: persistir accessKey, protocolNumber, submittedDocumentXml, responseXml, receivedAt, operationId
        }

        if ($result->isRejected()) {
            $failure = $result->failure();
            $policy = $result->retryPolicy();
            match ($policy?->mode) {
                RetryMode::AfterCorrection => /* remontar issue() com dados corrigidos + novo forOperation */,
                RetryMode::QueryBeforeRetry => /* byAccessKey() antes de reemitir */,
                RetryMode::AfterDelay, RetryMode::Immediate => /* filar conforme política */,
                RetryMode::Never, null => /* definitivo; não reenviar */,
            };
        }

        if ($result->isProcessing()) {
            // Host: consultar por chave/protocolo; não emitir de novo às cegas
        }
    }
}
```

## Handlers explícitos (avançado)

Use quando quiser despachar `IssueCteCommand` / Queries já materializados no container:

```php
use Suot\Fiscal\Cte\Application\Command\Issue\IssueCteCommand;
use Suot\Fiscal\Cte\Application\Command\Issue\IssueCteHandler;
use Suot\Fiscal\Core\Exception\InfrastructureException;
use Suot\Fiscal\Core\Result\RetryMode;

final class EmitirViaHandler
{
    public function __construct(
        private readonly IssueCteHandler $issueHandler,
    ) {}

    public function __invoke(IssueCteCommand $command): void
    {
        try {
            $result = $this->issueHandler->handle($command);
        } catch (InfrastructureException $e) {
            $failure = $e->toFailure('issue');
            // Host: alertar; filar retry conforme $failure->retryPolicy
            throw $e;
        }

        if ($result->isAuthorized()) {
            // Host: persistir accessKey, protocolNumber, submittedDocumentXml, responseXml, receivedAt, operationId
        }

        if ($result->isRejected()) {
            $failure = $result->failure();
            $policy = $result->retryPolicy();
            match ($policy?->mode) {
                RetryMode::AfterCorrection => /* $command->revise()->withOperationId(…)->…->build() + handle de novo */,
                RetryMode::QueryBeforeRetry => /* consultar por chave antes de reemitir */,
                RetryMode::AfterDelay, RetryMode::Immediate => /* filar conforme política */,
                RetryMode::Never, null => /* definitivo; não reenviar */,
            };
        }

        if ($result->isProcessing()) {
            // Host: consultar por chave/protocolo; não emitir de novo às cegas
        }
    }
}
```

Aliases úteis: `fiscal.cte`, `fiscal.cte.handler.issue`, queries `fiscal.cte.query.*`.

## Injetar `CteOs`

Mesmo padrão para CT-e OS (modelo 67):

```php
use Suot\Fiscal\CteOs\Application\CteOs;
use Suot\Fiscal\Core\Exception\InfrastructureException;
use Suot\Fiscal\Core\Result\RetryMode;

final class EmitirCteOsAction
{
    public function __construct(
        private readonly CteOs $cteOs,
    ) {}

    public function __invoke(): void
    {
        try {
            $result = $this->cteOs
                ->serviceStatus()
                ->forOperation('charter-status')
                ->forUf('TO')
                ->inHomologation()
                ->handle();
        } catch (InfrastructureException $e) {
            $failure = $e->toFailure('service_status');
            // Host: alertar; filar retry conforme $failure->retryPolicy
            throw $e;
        }

        if ($result->isOperational()) {
            // Host: seguir para $this->cteOs->issue()…->handle()
        }
    }
}
```

Aliases: `fiscal.cte-os`, `fiscal.cte-os.handler.issue`, `fiscal.cte-os.handler.cancel`, queries `fiscal.cte-os.query.*`.

Diagnóstico: `php artisan fiscal:cte-os:status --uf=TO`.

## Injetar `Mdfe`

Mesmo padrão para MDF-e (modelo 58):

```php
use Suot\Fiscal\Mdfe\Application\Mdfe;
use Suot\Fiscal\Core\Exception\InfrastructureException;
use Suot\Fiscal\Core\Result\RetryMode;
use Suot\Fiscal\Core\ValueObject\Uf;

final class EmitirMdfeAction
{
    public function __construct(
        private readonly Mdfe $mdfe,
    ) {}

    public function __invoke(): void
    {
        try {
            $result = $this->mdfe
                ->serviceStatus()
                ->forOperation('manifest-status')
                ->inHomologation()
                ->issuerUf(new Uf('RS'))
                ->handle();
        } catch (InfrastructureException $e) {
            $failure = $e->toFailure('service_status');
            // Host: alertar; filar retry conforme $failure->retryPolicy
            throw $e;
        }

        if ($result->isOperational()) {
            // Host: seguir para $this->mdfe->issue()…->handle()
        }
    }
}
```

Aliases: `fiscal.mdfe`, handlers `fiscal.mdfe.handler.*`, queries `fiscal.mdfe.query.*`.

Diagnóstico: `php artisan fiscal:mdfe:status --uf=RS`.

<Warning>
Não injete `Operation/*` (`IssueCte`, `IssueMdfe`, …) na aplicação — são `@internal`. Use `Cte`, `Mdfe`, `CteOs` ou Handlers públicos.
</Warning>

Diagnóstico CT-e: `php artisan fiscal:cte:status`. Detalhes no README do pacote `suot/fiscal-laravel`.
