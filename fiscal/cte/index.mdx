O CT-e (modelo 57) é o primeiro documento com API pública unificada no Fiscal SDK: client `Suot\Fiscal\Cte\Application\Cte` via `$cte->…->handle()`.

<Warning>
Capacidades fiscais só são production-ready após schemas importados, fixtures, ciclo de vida completo e evidência de homologação. Esta documentação descreve a API pública atual — não um selo de prontidão.
</Warning>
