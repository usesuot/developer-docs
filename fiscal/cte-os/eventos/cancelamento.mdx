## O quê / quando

CT-e OS autorizado que precisa ser anulado (evento `110111`). Irreversível. Justificativa com no mínimo 15 caracteres.

## Exemplo

```php
use Suot\Fiscal\Core\Exception\InfrastructureException;
use Suot\Fiscal\Core\Result\RetryMode;

try {
    $result = $cteOs->cancel()
        ->forOperation('ot-456-cancel')
        ->inHomologation()
        ->accessKey($chave44)
        ->emitterCnpj($cnpj)
        ->protocol($nProt)
        ->justification('Cancelamento solicitado pelo contratante após desistência do serviço.')
        ->handle();
} catch (InfrastructureException $e) {
    $failure = $e->toFailure('cancel');
    // Host: alertar; QueryBeforeRetry se timeout após envio — consultar antes de reenviar evento
    throw $e;
}

if ($result->isCancelled()) {
    // Host: persistir protocolNumber do evento, responseXml, accessKey, operationId; marcar documento cancelado
}

if ($result->isRejected()) {
    $failure = $result->failure();
    $policy = $result->retryPolicy();
    match ($policy?->mode) {
        RetryMode::AfterCorrection => /* corrigir justificativa/protocolo e tentar de novo */,
        RetryMode::QueryBeforeRetry => /* byAccessKey() antes de reenviar */,
        RetryMode::AfterDelay, RetryMode::Immediate => /* filar conforme política */,
        RetryMode::Never, null => /* definitivo; não reenviar */,
    };
}
```

Resultado: `CancelCteOsResult` — `isCancelled()`, `isRejected()`, `failure()`, `retryPolicy()`, `protocolNumber` do evento quando registrado.

Ver [Consulta por chave](/fiscal/cte-os/consultas/por-chave).
